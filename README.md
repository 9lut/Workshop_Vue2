# Vue E-Commerce & Admin Workshop

This project is a comprehensive Vue.js application that serves as both an E-Commerce storefront and an Admin Panel. It includes features for managing users, products, orders, and a complete shopping experience.

## Features

- **Authentication System:** Login, Registration, and Secure Routes.
- **Admin Panel:** Management interfaces for Users, Products, and Orders.
- **E-Commerce Storefront:** Product browsing, categories, and shopping cart functionality.
- **State Management:** Utilizing Vuex for global state (e.g., Cart, User Session).
- **Routing:** Vue Router for seamless navigation.
